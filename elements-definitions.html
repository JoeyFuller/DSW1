Elements and their definitions

<h1>
The main title heading, most often the biggest heading
<h2>
The second heading is often used for less important paragraph titles
<h3>
The third heading title if used to seperate paragraphs
<h4>
The fourth style header can be used as need along with the next few
<h5>
A fifth heading style
<h6>
A sixth heading style
<p>
The paragraph element tells the browser when to seperate each paragraph
<ul>
Creates a bulleted or unorder list
<ol>
This creates a list with a numbered structure
<li>
This creates a new point in a list, can be used with both <ol> and <ul>
<head>
This is the header for the entire document, contains files such as CSS and JS
<body>
This is where the contents of the page appear
<dl>
This creates a definitions list
<dt>
This shows a term to be defined
<dd>
This is the definition of the term used in the <dt>
<html>
Tells the broswer to render in HTML language
<title>
Labels the page, shows the title on tabs
<b>
This makes the text following bold
<i>
this gives an italic styling to any font inside the tags
<sup>
adds texts such as dates or prefixes
<sub>
adds characters below the line of text
<br>
adds a simple break in the page
<hr>
this creates a horizontal rule between two sections
<strong>
shows a text that is noticable outside of normal text to show importance
<em>
adds an italic style to one or more words to show emphasis on that certain word
