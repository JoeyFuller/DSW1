Elements and their definitions

<h1>
The main title heading, most often the biggest heading
<h2>
The second heading is often used for less important paragraph titles
<h3>
The third heading title if used to seperate paragraphs
<h4>
The fourth style header can be used as need along with the next few
<h5>
A fifth heading style
<h6>
A sixth heading style
<p>
The paragraph element tells the browser when to seperate each paragraph
<ul>
Creates a bulleted or unorder list
<ol>
This creates a list with a numbered structure
<li>
This creates a new point in a list, can be used with both <ol> and <ul>
<head>
This is the header for the entire document, contains files such as CSS and JS
<body>
This is where the contents of the page appear
